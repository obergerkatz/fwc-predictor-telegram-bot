# Prerequisites

## Required Software

- **Node.js**: 24.12.0 (managed via nvm)
- **pnpm**: 9.15.4+ (via corepack)
- **PostgreSQL**: 18+

## Required Credentials

- **Telegram Bot Token**: Obtain from [@BotFather](https://t.me/BotFather) on Telegram
- **football-data.org API Key**: Sign up at [football-data.org](https://www.football-data.org/) (free tier available)

## Note

When using Docker Compose (recommended for quick start), Node.js and PostgreSQL are automatically provided in containers. You only need Docker and Docker Compose installed.
