# Scoring Rules

## Match Predictions

| Prediction Accuracy | Points |
|-------------------|--------|
| Exact score match (e.g., predicted 2-1, actual 2-1) | 6 points |
| Correct goal difference (e.g., predicted 2-1, actual 3-2) | 4 points |
| Correct home/away score + correct result (e.g., predicted 2-1, actual 2-0) | 3 points |
| Correct home/away score only, wrong result (e.g., predicted 1-1, actual 1-2) | 1 point |
| No match | 0 points |

### Scoring Details

- If you predict one side correctly (home or away) AND the match result (win/draw/loss) correctly, you get 3 points
- If you predict one side correctly but the match result is wrong, you get 1 point
- Maximum 6 points per match
- Bets are scored based on 90-minute results only (extra time excluded)

### Examples

- Predicted 2-1, Actual 2-1 → 6 points (exact score)
- Predicted 2-1, Actual 3-2 → 4 points (both won by 1 goal difference)
- Predicted 2-0, Actual 2-1 → 3 points (correct home score + both teams won)
- Predicted 1-1, Actual 1-2 → 1 point (correct home score but predicted draw, actual away win)
- Predicted 2-1, Actual 0-0 → 0 points (no match)

## Tournament Predictions

### Top 4 Prediction

- Predict which teams will finish 1st, 2nd, 3rd, and 4th in the tournament
- Each correct position: 7 bonus points
- Maximum: 28 bonus points (if all 4 positions are correct)
- Must be placed before the first match starts
- Can modify individual positions or all at once before tournament begins

### Group Stage Prediction

- Predict the top 2 teams that will qualify from each group
- Each correct qualifier: 2 bonus points (position within group doesn't matter)
- Maximum: 48 bonus points (12 groups × 2 qualifiers × 2 points)
- Must be placed before the first match starts
- Can modify predictions for individual groups before tournament begins

### Bonus Points

- Tournament and group stage predictions earn bonus points that are added to your match prediction points
- Bonus points contribute to your total score on the leaderboard
- Predictions are locked once the first match of the tournament begins
